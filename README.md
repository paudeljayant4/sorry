# ❤️ Ultimate Apology Webpage for My Amazing Girlfriend

The most beautiful apology webpage ever created with a full 30-second animated experience!

## ✨ Features

- **Starry Night Background**: 200 twinkling stars with smooth animations
- **Floating Hearts**: Continuous heart animations floating up the screen
- **Crown Photo Frame**: Rotating, glowing frame with crown emoji
- **Beating Heart Animation**: Realistic heartbeat animation
- **30-Second Progressive Timeline**:
  - 0-4s: Title & Crown Photo
  - 4-8s: Heartfelt Apology Message
  - 8-12s: "I'M SO SORRY" with Blooming Roses
  - 12-18s: Four Promise Cards
  - 18-24s: Interactive Love Counter
  - 24-30s: Final "I Love You Forever" Message

- **Progress Bar & Timer**: Visual progress indicator
- **Interactive Elements**: Click to add love, sparkles on click
- **Glassmorphism Design**: Modern frosted glass effects
- **Responsive**: Works perfectly on mobile and desktop
- **Backend Ready**: Node.js/Express with Socket.io for real-time features

## 🚀 Quick Start

### Local Development
```bash
npm install
npm start
```
Then open http://localhost:3000 in your browser

### Deploy to GitHub Pages (Static Version)

1. Copy `public/index.html` to your GitHub repository
2. Enable GitHub Pages in repository settings
3. Your site will be live at `https://yourusername.github.io/repo-name`

### Deploy to Heroku/Railway/Render

1. Push this code to your Git repository
2. Connect to your hosting platform
3. The server will automatically start

## 💖 Customization

Edit `public/index.html` to personalize:
- Change the photo URL in the crown frame
- Modify the apology message text
- Adjust the promises
- Change colors and timing
